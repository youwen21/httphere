# 安装

## 使用go安装
go clone https://github.com/youwen21/httphere.git
or
go clone https://gitee.com/apiok/httphere.git

cd httphere

go mod tidy
go install


提供windows, linux, mocOS 三个平台可执行文件。

## 编译安装
```bash 
git clone http://gitee.com/youwen21/httphere
cd httphere 
go mod tidy
go install
```

# 启动HTTP服务
## 在任意目录执行命令  httphere  

```bash
httphere
```
