# 安装

## 使用go安装
go install gitee.com/youwen21/httphere

## 下载可执行文件
> https://gitee.com/youwen21/httphere/releases

提供windows, linux, mocOS 三个平台可执行文件。

## 编译安装
```bash 
git clone http://gitee.com/youwen21/httphere
cd httphere 
go mod tidy
go install
```

# 启动HTTP服务
## 在任意目录执行命令  httphere  

```bash
httphere
```
